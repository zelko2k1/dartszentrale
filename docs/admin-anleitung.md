# DartsHub – Admin-Anleitung: Inbetriebnahme & Updates

So bringst du die App zum Laufen und hältst sie aktuell — **ohne Vorkenntnisse**, getrennt nach
Betriebssystem. Öffne einfach die Anleitung für **dein** System:

| Dein System | Anleitung |
|---|---|
| 🪟 **Windows** (normaler PC/Laptop) | **[admin-anleitung-windows.md](admin-anleitung-windows.md)** |
| 🐧 **Linux / Raspberry Pi** | **[admin-anleitung-linux.md](admin-anleitung-linux.md)** |

Beide erklären dasselbe Schritt für Schritt:
- **Lokaler Modus** (ein Gerät, kein Server) — am einfachsten,
- **Vereinsmodus** (mehrere Geräte, Server im eigenen Netz),
- **Updates** einspielen, plus die wichtigsten Befehle und Hilfe bei Problemen.
